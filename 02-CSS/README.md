CSS (*Cascading Style Sheets*) is used to style and design webpages built with HTML.  
I am learning CSS step-by-step (following a structured video course) to build a strong foundation before moving toward JavaScript and advanced frontend technologies.

This journey is focused on:
- Understanding CSS syntax & selectors
- Styling real web pages
- Learning layout systems (Box Model, Flexbox, Grid)
- Making websites responsive
- Learning best practices
