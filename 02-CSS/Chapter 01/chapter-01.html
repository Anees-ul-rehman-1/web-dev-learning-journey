CSS

(Casceding Style Sheet)

It is a language that is used to describe the style of a document.

Basic Syntax

Selector

                    h1 {
                        color: red;
                       }
  • h1 is a Selector
  • color is a Property
  • red is a Value
  • and in css we use Curly brackets "{ }"

Including Style

01. Inline Style

                        <h1 style="color: red;">Anees Ul Rehman</h1>

Here we use style in same html inline tag.

03. <style> tag

                        <styel>
                               h1 {
                                   color: red;
                                  }
                        </style>

Here we use style tag in html head.

03. External Stylesheet

                    <link rel="stylesheet" href="style.css">

In this type we write CSS in seperate file (name.css), and then we link that file into HTML, head using link tag.

Color Property

Used to set the color of foreground

color: red;

color: pink;

color: blue;

color: green;

Background Color Property

Used to set the color of background

background-color: red;

background-color: pink;

background-color: blue;

background-color: green;

Color Systems

RGB

color: rgb(255, 0, 0);

color: rgb(0, 255, 0);



Hex (Hexadecimal)

color: #ff0000;

color: #00ff00;

Selectors

Universal Selector

                        * { }

Here asterisk/star "*" represnts Universal Selector which can style whole page.

Element Selector

                        h1 { }

Here h1 represnts all h1 elements which are present in linked html file.

Id Selector

                        #myId { }

Mostly we use id for only one element we don't use same id to multiple elemnts so id can only style that element.

Class Selector

                        .myClass { }

We can use one class to mulitple elements so far which those element has same class will get style by class.
