CHAPTER 2 – BASIC HTML TAGS

We can add elements inside the body tag to define the page layout.

HTML ELEMENT

An HTML element consists of everything from the starting tag to the ending tag.

    <body>
         content
    </body>

HTML ATTRIBUTES

HTML attributes provide additional information about an element.

        Example: 
        <a herf="https://www.google.com">Google</a>

We can either use single or double quotes in attributes.

THE HEADING TAG

Heading tags are used to define headings in HTML. There are six levels of headings, from <h1> to <h6>, with <h1> being the highest level and <h6> being the lowest.

Heading tag is used to mark headings in HTML. From h1 to h6, we have tags for the most important to the least important heading.

        For example: 
       
 Most important heading H1 

       
 Another heading H2 

       
 Another heading H3 
 
       
 Another heading H4 

       
 Another heading H5 
 
       
 Another heading H6

Note: We should not use HTML headings to make text thick or bold.

THE PARAGRAPH TAG

The paragraph tag is used to define a paragraph in HTML. It is represented by the <p> tag.

        For example: 
        <p> This is a paragraph. </p>

THE ANCHOR TAG

The anchor tag is used to create hyperlinks in HTML. It is represented by the <a> tag.

        For example: 
        <a href="https://www.google.com">Google</a>

THE IMAGE TAG

The image tag is used to insert images in HTML. It is represented by the <img> tag.

        For example: 
        <img src="image.jpg" alt="Description of image">

BOLD, ITALIC AND UNDERLINE TAGS

We can use the <b> tag to make text bold, the <i> tag to make text italic, and the <u> tag to underline text.

        For example:
        <b> This text is bold. </b>
        <i> This text is italic. </i>
        <u> This text is underlined. </u>

BR TAG

The <br> tag is used to insert a line break in HTML. It is an empty tag, meaning it does not have a closing tag.

        For example: 
        This is the first line. <br> This is the second line.

BIG AND SMALL TAGS

The <big> tag is used to make text larger, while the <small> tag is used to make text smaller.

        For example: 
        <big> This text is big. </big>
        <small> This text is small. </small>

HR TAG

The <hr> tag is used to insert a horizontal rule in HTML. It is an empty tag, meaning it does not have a closing tag.

        For example: 
        <hr>

SUBSCRIPT & SUPERSCRIPT TAGS

The <sub> tag is used to define subscript text, while the <sup> tag is used to define superscript text.

   
        For example: 
        H<sub>2</sub>O is the chemical formula for water.
        E = mc<sup>2</sup> is Einstein's mass-energy equivalence formula.

PRE TAG

The <pre> tag is used to define preformatted text. Text inside a <pre> element is displayed in a fixed-width font, and it preserves both spaces and line breaks.

        For example: 
        <pre>
        This is preformatted text.
        It preserves spaces and line breaks.
        </pre>
