In this chapter, we will learn about the basic HTML tags.

Ek HTML element sab kuch consist kar sakhta hia from start to ending tag. for example: body tag

Aur ek HTML attribute additional information provide karta hia ek element ke bare mein.

Heading tags total 6 levels ke hote hain, from h1 to h6. lkn hmm kabhi bhi text ko bold ya large karne ke liye heading tags use nhi kareenge q k as se site accessibility affected ho jati hain.

Paragraph tags HTML mein paragraphs define karne ke liye use kiye jate hain. Isko <p> tag se represent kiya jata hain.

Anchor tags HTML mein hyperlinks create karne ke liye use kiye jate hain. Isko <a> tag se represent kiya jata hain.

Image tags HTML mein images display karne ke liye use kiye jate hain. Isko <img> tag se represent kiya jata hain.

Bold tags HTML mein text ko bold karne ke liye use kiye jate hain. Isko <b> tag se represent kiya jata hain.

Italic tags HTML mein text ko italic karne ke liye use kiye jate hain. Isko <i> tag se represent kiya jata hain.

Underline tags HTML mein text ko underline karne ke liye use kiye jate hain. Isko <u> tag se represent kiya jata hain.

BR tag HTML mein line break insert karne ke liye use kiya jata hain. Isko <br> tag se represent kiya jata hain. Yeh ek empty tag hai, iska closing tag nahi hota.

Big and small tags HTML mein text ko big ya small karne ke liye use kiye jate hain. Isko <big> aur <small> tags se represent kiya jata hain.

hr tag HTML mein horizontal rule create karne ke liye use kiya jata hain. Isko <hr> tag se represent kiya jata hain.

subscript and superscript tags HTML mein text ko subscript ya superscript karne ke liye use kiye jate hain. Isko <sub> aur <sup> tags se represent kiya jata hain.

pre tag HTML mein preformatted text display karne ke liye use kiya jata hain. Isko <pre> tag se represent kiya jata hain.
