CHAPTER 1 – CREATING OUR FIRST WEBSITE

We start building a website by creating a file named index.html. index.html is a special filename which is presented when the website root address is typed.

A BASIC HTML PAGE

Here is a basic HTML page:

        <!DOCTYPE html>
        <html lang="en">

        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>My first webpage</title>
        </head>

        <body>
            <h1>My first webpage</h1>
            <p>This is my first webpage. I am learning HTML and it is really fun!</p>
            <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Quos, molestiae, saepe beatae, recusandae voluptatem
                dicta aspernatur impedit voluptates deserunt eos praesentium delectus consequuntur! Accusamus doloremque
                deleniti error iusto. Repellat natus totam eaque suscipit?</p>
        </body>

        </html>

A tag is like a container for either content or other HTML tags.

IMPORTANT NOTES

  • head & body tags are children of HTML tag.
  • HTML is the parent of head & body tags.
  • Most of the HTML elements have opening & closing tag with content in between opening & closing tags.
  • Some HTML tags have no content. These are called Empty elements e.g. br
  • We can either use .htm or .html extension.
  • you can use “inspect element” or “view page source” option from Chrome to look into a website’s HTML Code.

Comments in HTML

Comments are used to add notes in the code. These notes are not visible on the webpage. They are only for developers to understand the code better.

        <!-- This is a comment in HTML -->
        <!-- Comments are not visible on the webpage -->

Comments are very useful for explaining the code, especially when working in a team or when you want to remember why you wrote a certain piece of code in a particular way.

CASE SENSITIVITY

HTML tags are not case sensitive. This means that <h1> and <H1> are treated the same by the browser. However, it is a good practice to use lowercase letters for HTML tags to maintain consistency and readability in your code.

DOCTYPE DECLARATION

The <!DOCTYPE html> declaration is used to specify the version of HTML being used in the document. It helps the browser to render the page correctly. In modern web development, we use <!DOCTYPE html> to indicate that we are using HTML5, which is the latest version of HTML.
